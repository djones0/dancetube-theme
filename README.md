# AEDC Tube Jellyfin Theme

Custom Jellyfin CSS for the AEDC Tube brand — black background, red accents, graffiti-style logo.

## Usage

In Jellyfin → Dashboard → Branding → Custom CSS, paste:

```css
@import url("https://cdn.jsdelivr.net/gh/djones0/dancetube-theme@master/dancetube-theme.css?v=3.0");
```

Bump the `?v=` query when you update the theme so browsers and jsDelivr pick up the new file.
